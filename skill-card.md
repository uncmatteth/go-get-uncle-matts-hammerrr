# Go Get Uncle Matt's HAMMERRR

Hardcore Audit of Misleading Models, Evidence Reliability, Runtime, and Reality.

Use it when a repo's story stops matching system reality. HAMMERRR compares
claims against runtime behavior, config/deployment truth, tests, packaging,
release paths, and actual proof before anyone starts changing code.

Outputs `PROJECT_TRUTH_AUDIT.md`, with optional JSON when requested.
